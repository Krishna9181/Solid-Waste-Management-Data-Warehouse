# Solid-Waste-Management-Data-Warehouse
Designed, implemented, and queried a data warehouse in PostgreSQL for a solid waste management company operating hundreds of trucks across multiple cities in Brazil to analyze waste collection data.
Developed a star schema data warehouse with dimension and fact tables.
Generated analytical reports on waste collected by year, month, quarter, city, truck type, and station to support operational insights.
PostgreSQL, SQL, Data Modeling, Star Schema, ETL, Relational Database Design
